## WebIOT
Connect to and interact with your IoT devices over a variety of connections including:
- Serial
- USB
- Serial
- NFC
- More coming!

## Usage
### Installation
```
npm install @mastashake08/web-iot
```
### Usage Via Serial
```
import { SerialManager } from '@mastashake08/web-iot'

const sm = new SerialManager();

// get ports
sm.getPorts()
```
